# Asobiba Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** `あそびば` ハブページを追加し、既存の `図鑑` 導線を `top -> あそびば -> 図鑑` に変更する

**Architecture:** 既存の静的 HTML サイトに `asobiba.html` を追加し、通常ページのナビゲーションから直接 `monster-zukan.html` へ飛ばないようにする。`monster-zukan.html` 自体は既存の独自 UI を保ち、戻り先だけ `あそびば` に変更する。

**Tech Stack:** 静的 HTML, Tailwind CSS CDN, Vitest, Playwright
