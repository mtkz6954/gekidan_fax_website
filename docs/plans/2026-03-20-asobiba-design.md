# あそびばページ設計

## 目的

現在の `top -> 図鑑` 導線を `top -> あそびば -> 図鑑` に変更し、今後追加するゲームページの受け皿となる中間ハブをつくる。

## 方針

- ページ名とナビ表記は `あそびば` で統一する。
- ビジュアルは既存トップと連続した白ベースに寄せ、過度に図鑑寄りにはしない。
- 現時点では `ファニーモンスター図鑑` を主役にしつつ、`ゲーム` 用の `近日公開` 枠を並べる。
- `monster-zukan.html` は独立した遊びコンテンツとして残し、戻り先だけ `あそびば` に寄せる。

## ページ構成

1. ページヘッダー
   - タイトル: `あそびば`
   - 説明: 劇団FAXの遊べるページを集めた場所であることを短く示す
2. コンテンツカード
   - `ファニーモンスター図鑑`
   - `ゲーム` の近日公開カード
3. 補足メッセージ
   - 今後コンテンツが増えることをひとことだけ添える

## 導線変更

- `index.html`
- `about.html`
- `activity.html`

上記のヘッダーとモバイルナビで、従来の `図鑑` リンクを `あそびば` に差し替える。

## テスト方針

- 構造テストに `asobiba.html` を追加する。
- ナビゲーション期待値を `monster-zukan.html` ではなく `asobiba.html` に更新する。
- E2E では、ヘッダーナビから `あそびば` に遷移し、そこから `monster-zukan.html` に入れることを確認する。
