# 劇団FAX ウェブサイト — CLAUDE.md

## プロジェクト概要

劇団FAX（2016年京都旗揚げ）の公式ウェブサイト。次回公演『キルギスのやつ』の告知を中心に構成されている静的HTMLサイト。

## 技術スタック

- **Pure HTML** — ビルドツール・フレームワークなし（直接ブラウザで開ける）
- **Tailwind CSS** — CDN経由で読み込み（`https://cdn.tailwindcss.com`）
- **Google Fonts** — Noto Sans JP（400/500/700/900）、Press Start 2P（図鑑ページ用ピクセルフォント）
- **JavaScript** — 図鑑ページ（`monster-zukan.html`）でモンスター一覧表示・詳細切替・アニメーション制御に使用

## デザインシステム

### カラーパレット（Tailwind クラス）

**メインサイト（index / about / activity）**
- メインカラー: `sky-500` / `sky-600`（青系）
- アクセント: `orange-400` / `orange-500`（チケットCTA）
- テキスト: `slate-800`（本文）/ `slate-900`（見出し）/ `slate-500`（サブテキスト）
- 背景: `white`

**図鑑ページ（monster-zukan）**
- メインカラー: `purple-700` / `violet-500`（紫系、CSSカスタムプロパティ `--accent: #a78bfa` / `--accent2: #6d28d9`）
- 背景: `#0b0118`（ダークテーマ）
- テキスト: `slate-300`（本文）/ `white`（見出し）
- タイプ別バッジ色: 炎(orange) / 水(sky) / 草(green) / 闇(indigo) / 光(yellow) / 風(cyan) / 霊(purple) / 毒(violet)

### 共通UIパターン
- カード: `rounded-2xl border border-slate-100 p-6`
- CTA ボタン: `rounded-full bg-orange-400 px-7 py-3 text-sm font-black text-white`
- セクション幅: `mx-auto max-w-5xl px-5 md:px-8`
- ヘッダー: sticky、`bg-white/90 backdrop-blur-md`
- モバイルナビ: 画面下固定、5カラムグリッド

## 作業上の注意

- **ビルド不要**: HTMLファイルを直接編集するだけで反映される
- **Tailwind はCDN版**: カスタムクラスを `tailwind.config` の `extend` に追加できる
- **画像パスは相対パス**: `images/` からの相対参照で統一
- **日本語ファイル名の画像**: ファイル名にスペースや日本語を含む画像はHTMLでそのまま使える
- **モバイル対応必須**: 下部固定ナビがあるため `pb-16 md:pb-0` をmainに入れること
- **図鑑ページは独自テーマ**: 他ページと異なりダークテーマ・紫系配色。ヘッダー/フッターも図鑑専用デザイン
- **src/ は参考用**: `src/gekidan-fax.jsx` はReact版モックアップで、本番では使われていない
